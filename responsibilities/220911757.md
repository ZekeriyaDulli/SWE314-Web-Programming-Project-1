## Responsibilities

### Project Scaffolding & Configuration
- Initialised the React 18 + Vite project and configured `vite.config.js` with a dev proxy that forwards all `/api/*` requests to `http://localhost:8000`, eliminating CORS errors during local development
- Installed and configured Bootstrap 5 globally via `main.jsx`; defined the shared dark Glassmorphism design system (ambient gradient background, `backdrop-filter: blur` frosted-glass panels, crimson accent palette) used consistently across every page
- Set up the React Router v6 route tree in `App.jsx` with protected route wrappers that redirect unauthenticated users to `/login` and non-admins to `/`

### API Client & Authentication Layer
- Built `src/api/client.js` — a shared Axios instance with `baseURL` read from `VITE_API_URL` environment variable
  - Request interceptor reads `localStorage.getItem('token')` and injects `Authorization: Bearer {token}` on every outgoing request
  - Response interceptor catches `401` responses, clears the stored token, and redirects to `/login` to handle expired sessions gracefully
- Built `src/context/AuthContext.jsx` — global authentication state shared across the entire application
  - Exposes `user`, `token`, `isLoggedIn`, `isAdmin`, `login(tokenData)`, and `logout()`
  - `login()` persists the JWT and user object to `localStorage` so the session survives page refreshes and multiple browser tabs
  - `isAdmin` is derived by comparing `user.user_id` against the hardcoded `ADMIN_USER_ID` constant

### Core Pages
- Implemented `HomePage.jsx` — the main catalogue browsing page
  - On mount fetches `/shows/genres` for the genre dropdown and `/shows` for the initial unfiltered grid
  - Filter state (`genre_id`, `min_year`, `max_year`, `min_rating`, `search`) submitted as query params to `GET /shows`; only non-empty values are included
  - Client-side type filter (All / Movies / Series) applied on `show_type` without a server round-trip
  - **Mobile responsiveness**: on screens narrower than 768 px the five filter fields collapse behind a "⚙ Filters" toggle button (`filtersOpen` React state, default `false`); the type pills remain always visible; desktop layout is unaffected via Bootstrap `d-none d-md-flex`

- Implemented `ShowDetailPage.jsx` — the richest page in the application

  - Displays: proxy poster image, title, year, show type badge, runtime, IMDb rating (gold star), platform average rating with review count, full plot, genre chips, directors, cast

## Key Technical Decisions

- **Vite dev proxy**: the browser always communicates with `localhost:5173`; no CORS preflight issues during development and no URL changes between dev and production
- **JWT in `localStorage`**: session persists across tabs and refreshes; `get_optional_user` backend dependency means a single endpoint serves both anonymous and authenticated users
- **Pure React state for trailer modal**: no external modal library; avoids Bootstrap JS DOM conflicts and keeps the component fully declarative
- **Inline styles for design system**: prevents CSS class name collisions with Bootstrap; makes glassmorphism values explicit and easy to audit
