## Key Technical Decisions

- **`useRef` for polling interval**: holds the `setInterval` ID across re-renders without triggering re-renders itself; the `useEffect` cleanup function clears the interval when `AdminSyncPage` unmounts — eliminates the common memory-leak bug of leaving timers running after navigation
- **Optimistic state update on watchlist item removal**: the removed show is filtered from local state immediately on click rather than waiting for a refetch — makes the UI feel instantaneous
- **React state modals over Bootstrap JS**: delete confirmations use a `showConfirm` boolean state to toggle a custom modal; avoids direct DOM manipulation inside a React component and keeps the UI predictable
- **Admin guard in component body**: `if (!isAdmin) { navigate('/'); return null }` — admin pages self-eject non-admins before rendering any content, preventing even a flash of the admin UI
